Projects

Technical implementations and developments

Cryoscope Integration

A cryogenically cooled wide-field telescope designed for extreme Antarctic conditions. Engineered thermal control systems capable of operating at -80°C, with custom data acquisition pipeline using Telnet and USB SPI protocols. Implemented real-time monitoring with InfluxDB and Grafana dashboards for continuous system health checks.

Post-Earthquake Analysis

Developed machine learning models to analyze hydrological changes in earthquake-affected regions. Combined field data with SWAT and HEC-HMS simulations to predict water flow patterns. Created interactive visualizations of temporal changes in river dynamics and groundwater levels using Python and GIS tools.

Project Hriday

IoT-based cardiac monitoring system using Arduino and machine learning. Built a custom ECG signal processing pipeline with real-time anomaly detection. Implemented a TensorFlow model trained on 100,000+ heartbeat patterns, with automated report generation and emergency alert system via Gmail API.
